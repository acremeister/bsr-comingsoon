# Big Sandy Ranchettes Website

Static single-page property website for Big Sandy Ranchettes in Polk County, Texas.

## Files

- index.html - main production page
- images/ - all logo and photo assets
- images/IMAGE-PLACEHOLDERS.txt - required image filenames

## Local Preview

Open index.html in any browser.
